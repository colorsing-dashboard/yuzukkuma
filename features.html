ColorSing LP — Features

具体的に、何ができるのか

スプレッドシートに入力するだけで、リスナー向けの特典管理ページが自動で更新されます。各機能の詳細をご覧ください。

Ranking

前月ランキング表示

スプレッドシートに入力した歌推しポイントを読み込み、ランキング形式で自動表示。1〜3位は特別なデザインで強調表示されます。

  • ポイント順の自動ランキング表示
  • 1・2・3位のメダル特別表示（カスタム画像対応）
  • リスナー名クリックで個別詳細ポップアップ表示
  • 月別の履歴閲覧
スクリーンショット（準備中）
Benefits

特典管理

特典ティアごとの内容説明と、各リスナーの達成状況・受け取り履歴を一元管理。限定コンテンツのアクセスキー保護にも対応しています。

  • 特典ティア別の説明カード一覧
  • リスナーごとの達成ティア・特典履歴表示
  • アイコン・説明文のカスタマイズ
  • アクセスキーで保護した限定コンテンツ配信
  • テキスト・画像の限定コンテンツに対応

限定コンテンツ

アクセスキーを知るリスナーだけが閲覧できる、テキストや画像コンテンツを設定できます。

特典履歴

いつ・どの特典を受け取ったかをスプシで管理。リスナーページに自動反映されます。

スクリーンショット（準備中）
Icon Gallery

枠内アイコンギャラリー

枠内アイコンをギャラリー形式で一覧表示。月ごとにまとめてスプレッドシートで管理できます。

  • 月別のグリッドギャラリー表示
  • リスナー名の表示に対応
スクリーンショット（準備中）
Events

イベント告知・管理

次回イベントの告知カードをサイト上に自動表示。スプレッドシートにイベント情報を入力するだけで、リスナーへの告知が完了します。過去のイベント履歴表示にも対応。

  • 次回イベントの告知カード自動表示
  • 日程・タイトル・説明をスプシで管理
  • イベント履歴の一覧表示
  • 管理画面からイベントビューのON/OFF切り替え可能
スクリーンショット（準備中）
Customize

完全カスタマイズ

ヘッダー画像からカラーテーマ・フォント・エフェクトまで、40以上の設定項目を管理画面から変更できます。コードを書かずに、あなたの世界観を反映したサイトに仕上げられます。

  • PC / モバイル別のヘッダー画像設定
  • プライマリ・ハイライト等のカラーテーマ
  • フォント・タイトルスタイル（glass / gradient / plain）
  • グロー・フロートなどのエフェクト設定
  • 各ビューのON/OFF・表示順変更
スクリーンショット（準備中）
