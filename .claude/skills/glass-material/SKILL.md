# ガラス質感 — 物理ベース設計ルール

## 本物のガラスの特性

CSSの「glassmorphism」ではなく、実際のガラスの物理特性を再現する。

| 物理特性 | CSS表現 | 対応configキー |
|---------|--------|--------------|
| 素材の色味（ソーダガラスの鉄分による青緑） | radial-gradient + border色にティント | `brand.glassTint` |
| 表面反射（光源方向から斜めに映り込む） | linear-gradient 145deg 白→透明 | `brand.glassReflection` |
| スペキュラハイライト（光源の直接反射） | 上辺に細い白線 linear-gradient 90deg | `brand.glassSpecular` |
| 厚みによる端の不透明化 | inset box-shadow + radial-gradient | `brand.glassEdge` |
| 指向性ボーダー（光源側が明るい） | 4辺個別のborder-alpha | ティントから自動算出 |
| 背後のぼかし | backdrop-filter: blur | `brand.titleGlassBlur` |

## ライト/ダークテーマの違い

### ダークテーマ
- ティント色: `180,220,255`（明るい青）— 暗い背景に対して光る
- エフェクト値: そのまま使う
- 反射: 白のグラデーション

### ライトテーマ
- ティント色: `80,120,160`（暗い青灰色）— 白背景に対してコントラストを出す
- エフェクト値: **2.5倍に増幅**（白背景では弱い効果が見えない）
- 反射: 白のグラデーション（1.5倍強調）

**重要: ライトテーマで白ベースのティントを使うとガラスが見えない。必ず背景との明度差があるティント色を使う。**

## 推奨デフォルト値

```
glassTint: 0.08        (8%)   素材の色味
glassReflection: 0.75  (75%)  反射の強さ
glassSpecular: 0.95    (95%)  スペキュラ光沢線
glassEdge: 60          (60px) エッジの厚み
titleGlassBg: 0.35     (35%)  ガラス背景の濃さ
titleGlassBlur: 12     (12px) ブラーの強さ
```

## 実装場所

- レンダリング: `src/components/Header.jsx` の `TitleText` コンポーネント内 `effectiveStyle === 'glass'` ブロック
- デフォルト値: `src/lib/defaults.js` の `brand` セクション
- 管理画面: `src/admin/tabs/BrandingTab.jsx` の「素材感の詳細設定」アコーディオン

## ガラスが「ガラスに見えない」場合のチェックリスト

1. **背景がソリッドカラー1色** → `backdrop-filter: blur` は効果なし。背景に画像やグラデーションがあるか確認
2. **ライトテーマでティント色が白系** → コントラスト不足。暗い色（`80,120,160`等）を使う
3. **反射/スペキュラが弱すぎる** → ライトテーマでは値を大きくする
4. **エッジの厚みが小さい** → ガラスの存在感が薄い。30px以上推奨
5. **ボーダーが均一** → 実際のガラスは光源方向のエッジが明るい。4辺を個別に設定

## backdrop-filter: blur の限界

`blur()` は背後のコンテンツをぼかすだけ。ソリッドカラーをぼかしても同じソリッドカラーになる。
ガラス感の本質は blur ではなく:
- 反射（表面で光が跳ね返る）
- ティント（素材自体の色味）
- 厚み（エッジの存在感）
- スペキュラ（鋭い光の線）

これらはblur無しでもガラスに見える。blurはあくまで「すりガラス」の表現。
